# COE3200 - Communication Skills for Computing

## Introduction to Tools Used for Non-Verbal Communication

### Purpose of this Repository
This repository was created as part of the COE3200 assignment to demonstrate the use of GitHub as a non-verbal communication tool in the IT industry. It serves as a space to showcase basic coding skills and professionalism in managing code repositories.

### File Description: hello.py
The file `hello.py` is a simple Python script. When executed, it prints a greeting message to the console identifying the author (Tharindu Lakshan) and welcoming the user to the repository.
